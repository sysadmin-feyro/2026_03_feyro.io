# feyro.io Website

Marketing-Website fuer `feyro.io`, gebaut mit Vite, React, TypeScript und Tailwind CSS.

## Lokal entwickeln

```sh
npm install
npm run dev
```

Die App laeuft standardmaessig auf Port `8080`.

## Wichtige Skripte

```sh
npm run dev
npm run build
npm run lint
npm run preview
```

## Tech-Stack

- Vite
- React
- TypeScript
- Tailwind CSS
- shadcn/ui
- Supabase

## Umgebungsvariablen

Vorhanden bzw. optional genutzt:

- `VITE_SUPABASE_URL`
- `VITE_SUPABASE_PUBLISHABLE_KEY`
- `VITE_SUPABASE_PROJECT_ID`
- `VITE_GA_MEASUREMENT_ID`
